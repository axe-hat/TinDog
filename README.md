# TinDog - A Dating App for Dogs

A fun, responsive landing page for **TinDog** -- the Tinder-style dating app for dogs. Built with HTML5, CSS3, and Bootstrap 5.

## Features

- Animated gradient hero section with app download CTAs
- Feature highlights with Bootstrap icon components
- Testimonial section with press logos
- Three-tier pricing cards (Chihuahua, Labrador, Mastiff)
- Responsive footer with navigation links
- SVG favicon with paw-in-heart design
- Semantic HTML5 structure with accessibility enhancements
- CSS custom properties for easy theming
- Mobile and tablet responsive breakpoints

## Tech Stack

- **HTML5** -- Semantic elements (`header`, `main`, `section`, `footer`), ARIA attributes
- **CSS3** -- Custom properties, keyframe animations, media queries, smooth scroll
- **Bootstrap 5.3** -- Grid system, utility classes, responsive components

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/axe-hat/HTML-Animation.git
   cd HTML-Animation
   ```

2. Open `index.html` in your browser:
   ```bash
   # macOS
   open index.html

   # Linux
   xdg-open index.html

   # Windows
   start index.html
   ```

No build tools or dependencies required -- it's pure HTML and CSS with Bootstrap loaded via CDN.
